Place PWA icons in this folder:

- pwa-192x192.png (192x192 PNG)
- pwa-512x512.png (512x512 PNG)
- pwa-maskable-512x512.png (maskable 512x512 PNG)
- apple-touch-icon.png (180x180 PNG)
- favicon.svg (optional)
- favicon.ico (optional)

These are referenced by vite-plugin-pwa in vite.config.ts.
Replace these placeholders with your own branded assets.
